This is a basic web app that counts the number of words and characters in a given text.

## How to Run

1. Clone this repository
2. Install dependencies:
    ```
    pip install -r requirements.txt
    ```
3. Run the app:
    ```
    python app.py
    ```
4. Open your browser and go to:
    ```
    http://localhost:5000
    ```
